Python Script
1. Download and unzip Assignment 3
2. Navigate to the directory containing extract_words.py and run the script 
  python extract_words.py
3. The script should generate allwords.txt, uniquewords.txt, and wordfrequency.txt in the same directory. Ensure these exist.

MonoGame Program
1. Open Assignment 3 directory in your IDE
2. Ensure the generated .txt files and font file are located in the content directory and properly loaded by the Content pipeline
3. Build and run project (main executable is Game1)
The default view is the unique words visualizer and pressing enter toggles between the unique word display and word frequency display. Clicking the screen on the unique word display will display a new selection of random words.

DisplayUniqueWords:
Uses a chosen font, font size, and a palette of three colors to display randomly selected unique words with consistent spacing and no overlap.
DisplayWordFrequency:
Visualizes the relationship between word frequency and the number of words with that frequency using a custom graphical representation that highlights the distribution pattern commonly found in natural language text
